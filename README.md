# statsb


### Project setup
```bash
git clone https://github.com/statsb/statsb.github.io.git
cd statsb.github.io
yarn
```

### How to run on local
```bash
yarn dev
open http://localhost:3000
```

### How to deploy
```bash
yarn build

git add .
git commit -m "commit message"
git push origin main

yarn deploy
```
